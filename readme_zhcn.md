# _简单音频播放器_

------

个人实验性项目, 用ffmpeg/SDL和C语言写成. 还有不少缺陷, 仅供学习之用.
*如果不着急的话, 建议阅读ffmpeg的ffplay源代码.*
另外mpc-hc项目也是开源的, 也可以进行参考.

------

需要*Visual Studio 2015*以上版本
